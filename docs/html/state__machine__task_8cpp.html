Orders the robot arm to manipulate a block.

This function orchestrates the motion planning and execution for manipulating a block. It first calculates the inverse kinematics to reach the block's initial position. If the block is reachable, it proceeds to move the arm to reach the block, potentially in multiple steps. Then, it grasps the block and puts it in the desired order.

Parameters
nA reference to the ROS node handle.
bloccoThe block object representing the block to be manipulated.
iInteger that indicate the iteration of the loop that call this function.
Returns
A boolean value indicating whether the operation was successful, if it's true, the block was successfully manipulated, else the block was not reachable or other errors occurred.
