Orders the robot arm to manipulate a block. 

This function orchestrates the motion planning and execution for manipulating a block. It first calculates the inverse kinematics to reach the block\textquotesingle{}s initial position. If the block is reachable, it proceeds to move the arm to reach the block, potentially in multiple steps. Then, it grasps the block and puts it in the desired order.
